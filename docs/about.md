## Methodology

This textbook follows the **Intelligent Textbook** methodology developed by Dan McCreary, which structures educational content around:

- **Learning graphs** — 200+ concepts with explicit dependency relationships
- **Bloom's Taxonomy** — progressive learning objectives from Remember through Create
- **MicroSims** — interactive simulations for hands-on learning
- **Knowledge-first design** — concepts organized by prerequisite chains, not arbitrary chapter order

## How to Use This Book

1. **Linear reading** — Chapters build on each other. Start at Chapter 1.
2. **Reference lookup** — Use the [Glossary](glossary.md) and search for specific terms.
3. **Hands-on practice** — Each chapter includes practical exercises and MicroSims.
4. **Business planning** — Chapters 12-16 provide strategic frameworks for building a turf analytics practice.
